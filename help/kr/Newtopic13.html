조직관리

사업체 조직관리 화면 설명입니다.


일반사항

  • 사업체 조직 구성을 쉽게 파악 할 수 있습니다.
  • 기관/부서/직원 정보를  생성/수정/삭제할 수 있습니다.



1.  조직관리 화면

조직트리에서 각 항목을 누르면 해당 정보가 표시됩니다.



  • 구성 항목

항목

상세항목 

설명

버튼

기관등록

새 기관을 생성합니다.

부서추가

새 부서를 생성합니다.

조직트리에서 소속 기관/상위부서를 선택하고 부서추가 버튼을 누릅니다.

사원추가

새 직원을 생성합니다.

조직트리에서 소속 기관/소속 부서를 선택하고 사원추가 버튼을 누릅니다.

트리

조직트리

각 기관/부서/사원 항목을 누르면 오른쪽 정보보기란에 정보가 표시됩니다.



2. 기관정보 화면


  • 구성 항목

항목

상세항목 

설명

기관정보

기관코드

기관코드를 영문,숫자로 입력합니다.

기관명

기관 이름을 입력합니다.

메모

메모를 입력합니다.

사용여부

기관 사용여부(사용/사용안함)를 선택합니다.

버튼

저장

기관정보를 생성 또는 수정합니다.

삭제

기관정보를 삭제합니다.
