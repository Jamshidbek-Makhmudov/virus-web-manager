  1. VCS Kiosk Scanner

사용방법



1. VCS Kiosk Scanner 시작




2. 방문객 (방문신청 시 입력한) 전화번호 입력



2-1. 입력 전화번호 확인


- 방문신청 입력한 전화번호와 일치하는지 확인

- 사용자 확인을 위해 방문신청 전화번호와 동일하여야 한다.





3. 저장 장치 삽입


- 반입을 위해 검사 할 저장장치(HDD, USB 등)를 Kiosk Scanner에 결합한다.





4. 디스크 내역 확인


- 삽입한 디스크 내역과 동일한지 확인한다.

- 대용량 디스크인 경우 인식에 시간이 다소 걸릴수 있다.





4-1. 디스크 검사 시작


- 인식된 디스크를 바이러스 백신 검사 수행한다.

- 바이러스/악성코드가 검출되면 자동 치료 & 재검사를 수행

- 바이러스/악성코드가 검출되지 않으면 자동으로 결과 화면으로 이동





5. 디스크 검사 완료


- 디스크 검사가 완료되면 결합된 저장 매체를 분리한다.

- 검사 결과지가 자동으로 출력된다. 만약, 출력 되지 않으면 "결과 재출력" 버튼을 클릭하여 검사지를 출력한다.




5-1. 출력된 결과지 지참하여 저장 매체 반입


- 출력된 검사지를 지참하여 "반입 가능기한" 까지 기기를 반입해야 한다.
