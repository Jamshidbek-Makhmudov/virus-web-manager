점검현황

점검현황 화면 설명입니다.


일반사항

  • 일별/월별 점검현황을 확인 할 수 있습니다.
  • 기관별 점검현황을 확인 할 수 있습니다.
  • 부서별 점검현황을 확인 할 수 있습니다.
  • 취약점이 발견된 점검 현황을 확인 할 수 있습니다.
  • 악성코드가 발견된 점검 현황을 확인 할 수 있습니다.
  • 점검 결과에서 발견된 취약점 내용을 분석할 수 있습니다.
  • 점검 결과에서 발견된 악성코드 내용을 분석할 수 있습니다.

항목 

상세항목

설명

점검현황

일(day)

일별 점검 건수, 취약점이 발견된 점검 건수, 악성코드가 발견된 점검 건수를 차트로 표시합니다.

막대 그래프를 누르면 점검결과 페이지로 이동합니다.

월(month)

월별 점검 건수, 취약점이 발견된 점검 건수, 악성코드가 발견된 점검 건수를 차트로 표시합니다.

막대 그래프를 누르면 점검결과 페이지로 이동합니다.

기관별

점검현황


선택한 년,월의 기간별 점검현황을 표시합니다.

그래프를 누르면 점검결과 페이지로 이동합니다.

부서별 점검현황


선택한 년,월의 방문자 업무 담당 부서별 점검현황을 표시합니다. 그래프를 누르면 점검결과 페이지로 이동합니다.

보안 취약현황


선택한 년,월의 점검 결과 발견된 취약점 내용을 분석, 표시합니다.

악성코드 발견현황


선택한 년,월의 점검 결과 발견된 악성코드 내용을 분석, 표시합니다.
