관리자의 근무여부를 선택합니다.

관리자

등급

관리자 등급을 선택합니다.

최고 관리자 : 모든 권한을 가진 관리자 입니다.

기관 관리자 : 기관 권한만 가진 관리자 입니다.

보안관리자 :  점검결과 엑세스 권한을 가진 관리자입니다.

보안관리자(조회용) : 점검결과 조회 권한만 가진 관리자입니다.

관리자권한

메뉴권한

관리자가 사용할 WebManager 메뉴를 지정합니다.

관리기관

관리자가 관리하는 기관을 지정합니다.

관리검사장

관리자가 관리하는 검사장을 지정합니다.

버튼

목록

관리자 목록으로 이동합니다.

저장

입력한 내용을 저장하고 관리자 정보를 생성합니다.

초기화

입력한 내용을 지우고, 기본값 상태로 만듭니다.
