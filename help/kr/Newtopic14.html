직원관리

직원관리 화면 설명입니다.


일반사항

  • 직원 정보를 확인하고 생성/수정/삭제할 수 있습니다.
  • 직원의 직위/직책/직급 정보 등록을 위해서 직위/직책관리에서 사전 등록이 필요합니다.

2. 직원정보 등록 화면



  • 구성 항목

항목 

상세항목

설명

기본정보

사원명

사원의 이름을 입력합니다.

E-mail

사원의 E-mail을 입력합니다.

사번

사원의 사번을 입력합니다.

연락처

사원의 연락처를 입력합니다.

비밀번호

비밀번호를 입력합니다.

비밀번호 확인

입력한 비밀번호 확인을 위해 다시 동일한 비밀번호를 입력합니다.

상세정보

근무지

사원의 소속 기관을 선택합니다.

부서

사원의 소속 부서를 선택합니다.

직급

사원의 직급을 선택합니다.

직책

사원의 직책을 선택합니다.

직위

사원의 직위를 선택합니다.

사용언어

사원의 VCS 시스템 사용 언어를 선택합니다.

근무여부

사원의 근무여부를 선택합니다.

관리자

등급

사원의 관리자 등급을 선택합니다.

최고 관리자 : 모든 권한을 가진 관리자 입니다.

기관 관리자 : 기관 권한만 가진 관리자 입니다.

보안관리자 :  점검결과 엑세스 권한을 가진 관리자입니다.

보안관리자(조회용) : 점검결과 조회 권한만 가진 관리자입니다.

일반사용자 : 부여 권한이 없습니다.

관리자권한

메뉴권한

관리자가 사용할 WebManager 메뉴를 지정합니다.

관리기관

관리자가 관리하는 기관을 지정합니다.

관리검사장

관리자가 관리하는 검사장소를 지정합니다.

버튼

목록

직원 목록으로 이동합니다.

저장

입력한 내용을 저장하고 직원 정보를 생성합니다.

초기화

입력한 내용을 지우고, 기본값 상태로 만듭니다.
