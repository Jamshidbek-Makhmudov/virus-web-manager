    1. Introduction

    About VCS

    사업장내 반입되는 노트북, 저장매체의 바이러스 감염 여부를 사전에 점검/치료 하고, 최신 윈도우 업데이트를 유지 함으로써  

    반입된 저장매체로부터 바이러스 유입, 악성코드 등의 공격을 사전에 방지하는 솔루션입니다.


    - VCS Client : 방문전 반입할 노트북에 설치하여 사전 검사를 수행함으로써 바이러스 체크 및 Windows 업데이트로 보안 적합성 확보

    - VCS Kiosk Scanner : 방문시 저장매체(HDD, USB, CD-Rom 등) 검사 지원하며 각 저장 매체의 바이러스, 악성코드 감염여부를 검사
