업체별 점검현황

업체별 점검현황 화면 설명입니다.


일반사항

  • 업체별 점검현황을 확인 할 수 있습니다.
  • 업체의 일별/월별 점검현황을 확인 할 수 있습니다.
  • 업체 점검 결과에서 발견된 취약점 내용을 분석할 수 있습니다.
  • 업체 점검 결과에서 발견된 악성코드 내용을 분석할 수 있습니다.


1. 업체별 점검현황 화면




  • 검색 항목

항목 

상세항목

설명

점검기간

일자1,일자2

일자1~일자2 기간동안의 업체별 점검현황을 검색합니다.

반입기간

일자1,일자2

일자1~일자2 기간동안 반입처리된 점검건의 업체별  점검현황을 검색합니다.

점검구분

DOWNLOAD

방문자가 VCS Client 프로그램을 다운받아 설치하고 검사를 실행한 점검건의 업체별 점검현황을 검색합니다.

DPTKIOSK

사업체의 DPT 키오스크에서 검사를 실행한 점검건의 업체별 점검현황을 검색합니다.

KIOSK

사업체의 검사장 키오스크에서 검사를 실행한 점검건의 업체별 점검현황을 검색합니다.

장비구분

노트북

검사 장비가 노트북인 업체별 점검현황을 검색합니다.

저장매체

검사 장비가 저장 매체인 업체별 점검현황을 검색합니다.

진행상태

전체

전체 업체별 점검현황을 검색합니다.

점검완료

장비 검사가 완료되고  반입전인 점검건의 업체별 점검현황을 검색합니다.

반입완료

장비 검사가 완료되고  반입처리된 점검건의 업체별 점검현황을 검색합니다.

업체명


업체 이름으로 점검 현황을 검색합니다.


  • 구성 항목

항목 

설명

번호

목록의 번호를 표시합니다.

업체명

업체 이름을 표시합니다.

업체명을 누르면 아래에  업체의 점검현황이 표시됩니다.

점검현황

업체의 점검건수를 표시합니다.

악성코드발견

업체의 점검 중 악성코드가 발견된 점검건수를 표시합니다.

취약점발견

업체의 점검 중 취약점이 발견된 점검건수를 표시합니다.



2. 업체 점검현황 화면



  • 검색 항목

항목 

상세항목

설명

진행상태

전체

업체의 전체 점검 현황을 표시합니다.

점검완료

점검 완료하고 반입전인 업체의 점검 현황을 표시합니다.

반입완료

점검 완료하고 반입된 업체의 점검 현황을 표시합니다.

장비구분

장비전체

업체의 전체 점검 현황을 표시합니다.

노트북

업체의 노트북 점검 현황을 표시합니다.

저장매체

업체의 저장매체 점검 현황을 표시합니다.

일자

년(year)

선택한 년도의 업체의 점검 현황을 표시합니다.

월(month)

선택한 월의 업체의 점검 현황을 표시합니다.


  • 구성 항목

항목 

상세항목

설명

점검현황

일(day)

업체의 일별 점검 건수, 취약점이 발견된 점검 건수, 악성코드가 발견된 점검 건수를 차트로 표시합니다.

막대 그래프를 누르면 점검결과 페이지로 이동합니다.

월(month)

업체의 월별 점검 건수, 취약점이 발견된 점검 건수, 악성코드가 발견된 점검 건수를 차트로 표시합니다.

막대 그래프를 누르면 점검결과 페이지로 이동합니다.

보안 취약현황


선택한 년,월의 업체 점검 결과 발견된 취약점 내용을 분석, 표시합니다.

악성코드 발견현황


선택한 년,월의 업체 점검 결과 발견된 악성코드 내용을 분석, 표시합니다.
